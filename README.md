# SkillForge AI 🚀
### AI-Powered Skill Assessment & Personalized Learning Plan Agent

SkillForge AI is a career readiness diagnostic engine that goes beyond resume screening by evaluating actual proficiency, identifying skill gaps, and generating personalized learning roadmaps for job readiness.

## Problem
Resumes show what candidates claim to know, not how well they actually know it.

Candidates often struggle with:
- Knowing if they are truly job-ready
- Identifying real skill gaps
- Prioritizing what to learn next
- Translating effort into interview readiness

SkillForge AI bridges that gap.

---

## Features

### Intelligent Skill Assessment
- Job Description parsing
- Resume skill extraction
- Skill match and gap analysis
- Evidence-based proficiency scoring

### Adaptive AI Interview
- Conversational skill assessment
- Scenario-based technical probing
- Portfolio and interview readiness analysis
- Confidence vs demonstrated skill calibration

### Personalized Learning Plans
- 30 / 60 / 90 day readiness roadmap
- Weekly study execution plans
- Curated resources
- Project recommendations
- Skill ROI prioritization

### Career Readiness Diagnostics
- Readiness Score (/100)
- Skill Gap Heatmap
- Candidate Archetype Classification
- Interview Risk Forecast
- Hiring Manager Perspective

### Accountability Coaching
- Weekly progress tracker
- Sprint planning
- Consistency scorecards

## 6-Phase Framework

1. Intelligence Extraction
